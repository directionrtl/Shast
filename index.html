DemoGithubShast

With shast.js you can have a great hover effect on your images. (demo)

How to use

First, download the Shast plugin from github and include it in your html page.

Create a holder element with display:block and fixed width and height, then set image path in data-src on holder element, for example:
<a id='example' data-src='path_to_image.jpg'></a>
The best width and height is orginal image width and height. Also you can use width and height in percent.

Let's call the plugin: 

$(document).ready(function(){
	$("#example").shast();
});

Options

Option Default Description
scale 1.2 Scale rate for image.
vertical flase true or false ?
slices 3 Number of slices.
speed 500 Animation speed.
easing swing Easing name. (as you know if you want to use easing you need to include jQuery Easing Plugin)

Browser support

Almost all popular browsers!

Responsive?

Yes, it's responsive, if you use media queries or width in percentage, it'll reinit itself on resize. 

$(document).ready(function(){
	$("#demo_1").shast();
});
$(document).ready(function(){
	$("#demo_2").shast({
		slices: 5,
		speed: 300,
		easing: 'easeOutBounce'
	});
});
$(document).ready(function(){
	$("#demo_3").shast({
		vertical: true,
		slices: 4
	});
});
$(document).ready(function(){
	$("#demo_4").shast({
		scale: 1.1,
		speed: 1000
	});
});
